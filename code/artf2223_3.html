Cuisine du Covid

2020

❮ Stay home and dress up
New Age Love ❯

A satirical website created for 5D Fundamental's Tactical Media Project, in which groups of students were tasked with designing a website which hosts a tactical media intervention. We chose to create a website for a fictional high-class pandemic-themed restaurant in response to the disconnect between the ultra-rich and the reality of the coronavirus. I designed the logo and coded the website, which can be visited here.
